# AppDost — Remade Version

This project is a redesigned and improved version of the original [AppDost platform](https://appdost.in/).
It keeps the same structure and layout as the original website but introduces a modern and refreshed user interface with glassmorphism effects, light/dark mode, and a sign-in popup.

## Features

* Glassmorphism-based UI
* Light and dark mode toggle
* Sign-in popup
* Smooth scrolling and redirection
* Responsive and clean layout

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

1. Open the project folder in VS Code.
2. Open the `index.html` file.
3. Go to **Run → Run Without Debugging** to view it in your browser.
4. OR else visit the deployed link : https://appdosti.netlify.app/
## Note

This is not a copy or clone of the original website.
It is a remake with a fresh design and improved user experience inspired by AppDost.
